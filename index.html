SeanPreme eCommerce

Demo Code

• Node.js • Postgres• Express • Sequelize • React • Chakra UI • Redux • Redux-Thunk • AWS • Docker • Docker Compose • Nginx • Surge

A fashion eCommerce store built using the PERN stack. Node.js and Express were primarily used for the backend. Postgres and Sequelize were used as the database and ORM respectively. The frontend was built using React, Redux, and Chakra UI. Third party APIs include Google OAuth for authentication and Stripe.js for online payments. The backend was containerized with Docker and Docker Compose and deployed to an AWS EC2 server using Nginx as a reverse proxy. Frontend was deployed using Surge.

BlackJack

Code

• Redis • Postgres• Express • Sequelize • React • Chakra UI • AWS • Docker • Surge

A simple blackjack game app built using the PERN stack. The backend was built using Node.js and Express. Authentication and authorization are handled through local and JWT with Passport. Redis was used to store temporary data such as game session information while Postgres was used to store permanent data such as account info. Sequelize was used an the ORM. The frotend was build using React and Chakra UI. The frontend was deployed using Surge and the server is containerized with Docker and hosted on AWS.
