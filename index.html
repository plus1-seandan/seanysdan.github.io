BlackJack

Demo Code

• Redis • Postgres• Express • Sequelize • React • Chakra UI • AWS • Docker • Surge

A simple blackjack game app built using the PERN stack. The backend was built using Node.js and Express. Authentication and authorization are handled through local and JWT with Passport. Redis was used to store temporary data such as game session information while Postgres was used to store permanent data such as account info. Sequelize was used an the ORM. The frotend was build using React and Chakra UI. The frontend was deployed using Surge and the server is containerized with Docker and hosted on AWS.

Youtube Sync

Demo Code

• Socket.io • Postgres• Express • Sequelize • React • Chakra UI • Material UI • Redux• AWS • Docker • Surge

An app allowing you to view videos together and chat about them in realtime. The backend was built using Node.js and Express. Authentication and authorization are handled through local and JWT with Passport. Implemented websockets using Socket.io. The frotend was build using React, Chakra UI and Material UI. Global states were managed with Redux. Postgres and Sequelize were used as the database and ORM respectively. The frontend was deployed using Surge and the server is containerized with Docker and hosted on AWS.
